2. What are the css precedence rules and css specificity?

Css specificity is the set of rules applied to determine which style is applied to an element. Order of css specificity from highest to lowest is as follows:-

  1. A css rule with !important always takes precedence.
  2. If element is having inline style ( element style attribute ) overrides all other styles (1,0,0,0)
  3. css selector using element id (element id attribute) (0,1,0,0)
  4. css selector using class, pseudo class or attribute (0,0,1,0)
  5. css selector using element name (for e.g. h1, p, div) (0,0,0,1)
  6. css selector appear later in the code override earlier one if both have the same specificity Good read to understand how to calculate specificity:- https://dev.to/emmabostian/css-specificity-1kca
